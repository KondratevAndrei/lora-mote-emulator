LoRa Motes Emulator
===================

This is a useful tool to test LoRa server.

To emulate end devices (AKA Motes in |LoRaWAN(TM)| protocol)

*Support LoRaWAN 1.0.2 protocol*

**Using Gateways from |Semtech(TM)|**

.. |Semtech(TM)| unicode:: Semtech U+2122
